Comment obtenir un Client ID Google ? ›
  1. Aller sur console.cloud.google.com
  2. Créer un projet (ou sélectionner un projet existant)
  3. APIs & Services → Bibliothèque → activer Google Drive API
  4. APIs & Services → Écran de consentement OAuth → Type : Externe → Statut : Test → Ajouter votre email comme utilisateur test
  5. Dans l'écran de consentement → Ajouter ou supprimer des champs d'application → cocher https://www.googleapis.com/auth/drive
  6. Identifiants → + Créer identifiants → ID client OAuth 2.0
  7. Type d'application : Application Web
  8. Dans « Origines JavaScript autorisées » : ajouter
  9. Copier le Client ID et le coller ci-dessus
  10. ⚠️ Si vous avez déjà un token, cliquez Déconnecter puis reconnectez-vous pour obtenir les nouveaux scopes
